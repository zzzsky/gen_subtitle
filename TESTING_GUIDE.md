✅ 任务列表显示
✅ 新任务出现在列表中
✅ 显示任务信息：

   - 模型名称

## 测试场景 6: 状态自动转换

### 测试转换矩阵

| 当前状态 | 操作 | 预期结果 |
|---------|------|---------|
| Idle | 导入文件 | → Processing |
| Processing | 删除所有任务 | → Idle |
| Processing | 选中已完成任务 | → Editing |
| Editing | 取消选择任务 | → Processing |
| Editing | 删除所有任务 | → Idle |

### 测试步骤
1. **Idle → Processing**
   - 确保在 Idle 状态
   - 导入一个视频文件
   - ✅ 应该切换到 Processing

2. **Processing → Idle**
   - 在 Processing 状态
   - 删除所有任务（右键任务 → Delete）
   - ✅ 应该切换回 Idle

3. **Processing → Editing**
   - 在 Processing 状态
   - 选中一个 Completed 状态的任务
   - ✅ 应该切换到 Editing

4. **Editing → Processing**
   - 在 Editing 状态
   - 点击 "返回列表" 按钮
   - ✅ 应该返回 Processing

## 常见问题排查

### 应用无法启动
```bash
# 检查构建
dotnet build src/GenSubtitle.App/GenSubtitle.App.csproj

# 检查依赖
# 确保安装了 .NET 7.0 SDK
dotnet --list-sdks
```

### 视频无法播放
- 确保视频文件路径有效
- 检查是否安装了必要的视频编解码器
- 尝试不同的视频格式（.mp4 通常是最佳选择）

### 任务状态不变
- 检查是否配置了 Whisper 和 Qwen API
- 查看日志输出确认错误信息
